活动首页 - 需求文档

页面基本信息

页面名称 活动首页
路由路径 /app/home.html
所属端 小程序端（C端）
优先级 P0 - 核心功能
页面类型 列表页
屏幕适配 最大宽度 375px，居中显示

1. 功能概述

活动首页是用户端的核心入口，用于展示所有可参与的活动信息，支持活动分类筛选、搜索和快速参与。

2. 页面结构

2.1 页面头部
包含页面标题、副标题和搜索栏。
  • 标题: "精彩活动"，白色字体，渐变背景
  • 副标题: "发现更多有趣活动"，透明度0.8
  • 搜索栏: 半透明背景，圆形输入框
2.2 Banner轮播
顶部活动宣传图，支持手动滑动切换。
  • 高度: 120px
  • 圆角: 8px
  • 左右间距: 16px
2.3 分类Tab栏
固定在顶部，支持以下分类切换：
  • 全部
  • 进行中
  • 即将开始
  • 已结束
2.4 活动卡片列表
展示活动信息，包含：
  • 封面图: 高度160px，圆角12px
  • 活动标题: 16px，加粗
  • 活动时间: 12px，灰色
  • 标签: 热门(红色)、新上线(绿色)
  • 底部: 参与人数 + 参与按钮
2.5 底部导航栏
固定在底部，包含TabBar导航。
  • 首页 - 当前选中
  • 我的 - 个人中心入口

3. 交互说明

3.1 搜索功能

[!IMPORTANT] 搜索规则
  • 点击搜索栏进入搜索页面
  • 支持按活动名称关键词搜索
  • 搜索历史记录保留最近10条

3.2 Tab切换

[!IMPORTANT] 切换规则
  • 点击Tab立即切换，无动画
  • 切换时保留滚动位置
  • 当前选中Tab下方显示蓝色指示条

3.3 活动卡片

[!IMPORTANT] 点击行为
  • 点击卡片跳转到活动详情页
  • 点击"立即参与"按钮直接跳转
  • 已结束活动显示"查看详情"按钮

4. 状态说明

状态 显示样式 说明
进行中 正常展示，按钮显示"立即参与" 活动正在进行中
即将开始 显示倒计时标签 活动尚未开始
已结束 按钮显示"查看详情"，卡片置灰 活动已结束

5. 接口设计

获取活动列表
GET /api/activities

参数:
- category - 分类筛选
- keyword - 搜索关键词
- page - 页码
- pageSize - 每页条数
获取Banner
GET /api/banners

参数:
- position - 位置标识

6. 异常处理

  • 网络错误: 显示错误提示和重试按钮
  • 数据为空: 显示空状态插画
  • 加载中: 显示骨架屏
  • 下拉刷新: 支持PullToRefresh
