## 📋 使用指南

### 何时使用此 Mock 数据库

1. **项目类型判断**：
   - 美妆电商、护肤商城 → ✅ 使用此数据库
   - 品牌官网、会员中心 → ✅ 使用此数据库
   - 积分商城、兑换系统 → ✅ 参考产品信息
   - 其他类型项目 → ⚠️ 按需选择

2. **智能体使用步骤**：
   1. 确认项目是否涉及美妆/护肤领域
   2. 如果是，优先使用此数据库中的产品数据
   3. 根据具体项目需求，选择合适的品牌和产品线
   4. 生成 Mock 数据时，确保数据真实性和合理性

3. **数据组合建议**：
   - **美妆电商**：产品库 + 购物车 + 订单 + 优惠券 + 评价
   - **品牌官网**：产品库 + 会员信息 + 订单
   - **积分商城**：产品库（作为兑换商品）+ 积分/余额 + 订单
   - **会员中心**：用户信息 + 积分/余额 + 订单 + 优惠券
